#  Sentiment Analyzer

A Python-based **Sentiment Analysis** project that classifies text as **positive, negative, or neutral**. This project demonstrates basic Natural Language Processing (NLP) concepts using Python.

---

## Features
- Text sentiment classification
- Preprocessed input text
- Generates output results
- Simple and modular project structure

---

##  Tech Stack
- Python
- NLP libraries (as listed in `requirements.txt`)
- Virtual environment support
